# typescript-methods-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- Why do we log things to the console? its a debugging tool and allows you to check data

- What is a method? a method is a function that is a property of an object

- How is a method different from any other function? functions dont have to be related to objects but methods have to be attached to objects

- How do you remove the last element from an array? pop() method

- How do you round a number down to the nearest integer? by using the Math.floor() method

- How do you generate a random number? Math.random() method

- How do you delete an element from an array? splice()

- How do you append an element to an array? by using the unshift() method which add it to the front of the array and push() which adds it to the back of an array

- How do you break a string up into an array? by using the split() method

- Do string methods change the original string? How would you check if you weren't sure? no strings are immutable and to check you can log the variable to the console

- Is the return value of a function or method useful in every situation? no

- How do you get a collection of an objects keys? by using the object.keys() method of the objects object

- How do you get a collection of an objects values? by using the objects.values() method of the objects object

- How do you get a collection of both an objects keys and values? by using the objects.entries() method of the objects object
